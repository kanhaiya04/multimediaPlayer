# Multimedia Player

 Multimedia player which can display image, gif, and video.

 [![Generic badge](https://img.shields.io/badge/view-demo-blue?style=for-the-badge)]()



# Run locally

**Clone repo**
```bash
git clone 
cd multimediaPlayer/
```
**Use npm to install packages**

```bash
npm install
```

**Run on hostlocal host 3000**
```bash
npm start
```
